# Vansh Saxena Portfolio Website

## Overview

A modern, responsive personal portfolio website showcasing the profile,
skills, projects, experience, coding achievements, and contact
information of Vansh Saxena. The portfolio is designed with premium UI
effects, smooth animations, custom interactions, and an AI-inspired
visual theme.

------------------------------------------------------------------------

## Features

### User Interface

-   Modern and responsive design
-   Custom animated cursor
-   Interactive loading screen
-   Mobile-friendly navigation menu
-   Smooth scrolling experience
-   Scroll-based reveal animations
-   Professional hero section with profile image

### Interactive Components

-   Dynamic role rotator
-   Neural-network animated background using HTML5 Canvas
-   Animated counters and statistics
-   GitHub-style contribution heatmap
-   Contact form with email integration
-   Live LeetCode statistics integration

### Performance & UX

-   Optimized animations
-   Visibility-aware canvas rendering
-   Responsive layouts for desktop, tablet, and mobile
-   Lightweight implementation using Vanilla JavaScript

## Installation

### Clone Repository

``` bash
git clone <repository-url>
```

### Open Project

``` bash
cd vansh-portfolio
```

### Run Locally

Simply open:

``` text
index.html
```

or use a local development server:

``` bash
npx serve
```

------------------------------------------------------------------------

## Customization

### Update Personal Information

Edit:

``` text
index.html
```

Update: - Name - Bio - Education - Experience - Social links

### Change Profile Image

Replace:

``` text
assets/photo.jpeg
```

with your own image.

### Update LeetCode Username

Inside:

``` javascript
js/leetcode.js
```

Modify:

``` javascript
const USERNAME = "your_username";
```
